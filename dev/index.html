Interfaces.@implements — Macro
@implements(interface, objtype, obj)

Declare that an interface implements an interface, or multipleinterfaces.

Also pass an object or tuple of objects to test it with.

The macro can only be used once per module for any one type. To define multiple interfaces a type implements, combine them in square brackets.

Example

@implements BaseInterfaces.Iteration{(:indexed,)} MyArray MyArray([1, 2, 3])

Interfaces.@interface — Macro

@interface

Define an interface.

@interface MyInterface begin
    mandatory = (
        length = x -> length(x) = prod(size(x)),
        ndims = x -> ndims(x) = length(size(x)),
    ),
    optional = (;)
end
